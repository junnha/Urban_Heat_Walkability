# Urban_Heat_Walkability

LH COMPAS 폭염 대응 보행환경 분석 작업현황
- 팀명: 천곡소년단
- 팀장 : 박기환
- 팀원 : 박의성, 전지수, 황준하
- 작업 기준일: 2026-07-07

<br>

## 프로젝트 개요
- 본 프로젝트는 LH COMPAS 공모전 「폭염 대응 우선관리지역 도출을 통한 더위회피 보행 동선 및 스마트 폭염대응 방안 제안」 제출을 목표로 한다.

1. 30m 격자 단위로 도시 열취약도를 산정한다.
2. Landsat 기반 지표면온도(LST)를 반영하여 최종 열취약도 지표를 만든다.
3. 실제 보행 도로망이 제공된 판교·분당 지역을 대상으로 보행 열노출 및 더위회피 곤란구간을 분석한다.
4. 왕숙1·창릉은 개발 중인 3기 신도시이므로, 격자 기반 최종 열취약도 결과를 활용하여 향후 스마트 폭염대응시설 배치 방향을 제안한다.

분석 방향은 다음과 같이 정리한다.

- 판교·분당: 실제 도시 / 상세도로망 기반 보행 네트워크 상세분석
- 남양주 왕숙1, 고양 창릉: 계획도시 / 격자 기반 열취약도 및 계획 시사점 분석

<br>

- 작업기간 : 2026.07 ~ 2026.08
- 분석대상지 : 판교, 분당
- 적용대상지 : 남양주 왕숙1, 고양 창릉
- 공간분석단위 : 30m × 30m Grid
- 스킬 및 사용툴 `Python` `QGIS` `GeoPandas` `NetworkX` `Scikit-learn` `OpenCV`

<br>

## 문제 정의

- 기후변화로 인해 도시 내 폭염 빈도와 강도 증가
- 아스팔트, 콘크리트, 고밀도 건축물 중심의 도심 지역은 열이 쉽게 축적됨
- 기존 폭염 대응은 무더위쉼터, 그늘막 등 개별 시설 확충 중심
- 실제 보행자는 생활권 동선에서 이동 중 열환경에 지속적으로 노출됨

→ 목표 : 열환경, 공간구조, 열저감 여건, 대응시설 접근성을 종합하여 보행자가 폭염 상황에서 더위를 피하기 어려운 구간을 도출

<br>

## 지표 설정

- 열환경 지표
  - 30m × 30m 격자별 열환경 강도
  - 지표면온도 및 고온 격자 분포

- 공간구조 지표
  - 건물 밀도
  - 도로 밀도
  - 불투수면 비율
  - 보행 네트워크 구조

- 열저감 여건 지표
  - 녹지율
  - 공원 접근성
  - 수변공간 접근성
  - 그늘 가능성

- 대응시설 접근성 지표
  - 무더위쉼터 거리
  - 그늘막 거리
  - 쉘터형 정류장 접근성

- 경로 분석 지표
  - 최단경로 거리
  - 더위회피 경로 거리
  - 우회율
  - 열노출 감소율

<br>

## 컴퓨터 비전 활용

- 항공영상 또는 위성영상 기반 보행환경 요소 추출
- Semantic Segmentation을 활용한 녹지, 도로, 건물, 그림자 영역 분리
- 격자별 녹지율, 그늘율, 불투수면 비율 산정
- CV 기반 지표를 보행환경 열 스트레스 지수에 반영

→ 지표면온도만으로 파악하기 어려운 보행자 체감 더위회피 가능성을 보완

<br>

## 실험 진행

1. 판교·분당 30m × 30m 격자 단위 열환경 데이터 구축
2. 보행로 및 도로 네트워크 데이터 구축
3. 건물, 녹지, 공원, 대응시설 등 공간정보 결합
4. 컴퓨터 비전 기반 녹지·그늘·불투수면 지표 추출
5. 공간분석단위별 보행환경 열 스트레스 지수 산정
6. 주요 생활권 동선 선정
7. 최단경로와 더위회피 경로 산출
8. 경로별 열노출 수준, 우회율, 열노출 감소율 비교
9. 더위회피 곤란구간 도출

<br>

## 결과 분석

- 폭염 취약지역 지도화
- 생활권 동선별 최단경로 / 더위회피 경로 비교
- 열노출이 높고 대체경로 선택이 어려운 더위회피 곤란구간 도출
- 취약 원인을 열저감 여건, 대응시설 접근성 측면에서 해석
- 그늘막, 가로수, 무더위쉼터 등 폭염 대응시설 설치·개선 우선지역 제안
- 3기 신도시의 폭염 대응형 보행환경 설계 시사점 도출
